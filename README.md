# Proyecto App de Delivery

## Descripción

Proyecto de base de datos desarrollado en SQL para administrar las operaciones de una plataforma de delivery.

El sistema permite gestionar clientes, restaurantes, repartidores y pedidos, manteniendo la integridad de la información mediante relaciones entre tablas y restricciones de base de datos.

## Tecnologías utilizadas

- SQL
- MySQL

## Funcionalidades

- Gestión de clientes
- Gestión de restaurantes
- Gestión de repartidores
- Registro de pedidos
- Seguimiento de pedidos
- Consultas mediante vistas SQL
